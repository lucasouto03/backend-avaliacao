# Backend Avaliação

API desenvolvida em TypeScript para gerenciamento de usuários e autenticação com JWT.

## Tecnologias

* TypeScript
* Express
* Prisma
* SQLite
* JWT

## Como executar

```bash
npm install
npm run dev
```

A aplicação será executada em:

```txt
http://localhost:3000
```

## Funcionalidades

* CRUD de usuários
* Login com JWT
* Proteção de rotas com autenticação
